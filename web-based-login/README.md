# Web-Based Login Authentication

A simple, client-side login page built with vanilla HTML, CSS, and JavaScript. The app validates user input in the browser and checks credentials against hardcoded values—no backend, database, or external dependencies required.

## Project Overview

This project demonstrates a basic authentication flow using only static files:

- A responsive login form centered on the screen
- Client-side validation for empty username and password fields
- Success and error feedback after login attempts
- A reset button that clears the form and messages
